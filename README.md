# CSO Redirect
Small plugin that redirects files based on lists of wildcard/files from CSO to drive folder.

## Usage
- You will need create file under plugins folder with name `CSORedirect.list`.
  - This file will contain file list with full path that you will want to redirect from CSO.
  - You can use wildcard to redirect all files inside folder as `script\*` or `script\*.cso`.
